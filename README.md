# csv-tranposer

## これは何ですか？
ブラウザの中で、
CSVファイルまたはクリップボードからペーストされたTSVデータに対し、
転置（行と列の入れ替え）を行うものです。

変換結果はテキストエリアで確認できます。

この結果はCSVファイルとしてダウンロードするか、
TSVとして（あるいはCSVとして）クリップボードにコピーできます。

ダウンロードしたCSVファイルはBOMが付与してあるので、
Microsoft Officeで直接開くことができます。

## 誰が必要としますか？
Excelには「形式を選択して貼り付け」の中に転置貼り付けの機能があり、
また、Excel関数にも```WorksheetFunction.Transpose```メソッドがあるので、
Excelに十分に習熟している人、または習熟しようとしている人には必要ありません。

しかしトレーニング中の初学者が頻繁に陥る状況として
1. サンプルファイル群をダウンロードするとzipファイルになっている
2. zipファイルを展開せずにダブルクリックして作業をはじめる
3. 転置貼り付けの機能が抑止されている

というものがあります。

* zipファイルを展開してその中身のファイルを操作する
* zipファイルを展開せずエクスプローラの一時展開ファイルを開く

の区別ができないWindowsユーザがあまりにも多いため、これを作りました。

## 情報漏洩になりませんか？
このプログラムはローカルPCにHTMLファイルを置いてwebブラウザで開けばオフラインでも動作します。

## インタフェースが古臭いのですが、もっとモダンになりませんか？
HTMLファイルをローカルに置けば動作するものにしたいので、
Webフレームワークは使わない方針となっております。

カスタマイズは制限しないライセンスとしておりますので、
必要に応じて御修正いただけますようお願いいたします。
